Jarvis preferences

Jarvis can notify you when a note’s time is due. Browser notifications require this tab to be open; Telegram delivers to your phone regardless.

Setup:
1. Open Telegram, search @BotFather, send /newbot
2. Copy the bot token and paste above
3. Search @userinfobot, send /start to get your Chat ID
4. Important: Send any message to your new bot first so it can reply to you
With cloud sync on, a scheduled job in Supabase can send this via Telegram at your chosen time even when the app is closed (see supabase/functions/morning-brief). Otherwise the brief only runs while this tab is open.
